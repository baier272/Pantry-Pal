# 食材管理

## 如何使用
- freshfood（“食材精”） 是主程序，可直接单独使用。
- menu（“菜谱精”） 可以和 freshfood 结合使用。
- 首次使用 menu 前需要先运行过1次 freshfood（menu 中调用的两个数据文件需要在 freshfood 中初始化。

## freshfood 的主要功能：
- 添加、删除食材类别（如：蔬菜、蛋白质、碳水）
- 按类别添加、删除食材
- 增减、减少食材数量
- 将食材转移到“危险“（临期）列表

## menu 的主要功能：
- 记录菜式需要的食材
- 从 freshfood 数据中查询要做某个菜式还缺什么食材
